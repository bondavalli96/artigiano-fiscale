## Integration in the App

### 1. Host on Your Website

Create pages:
- `https://artigianoai.it/terms` (IT)
- `https://artigianoai.it/en/terms` (EN)
- `https://artigianoai.it/es/terms` (ES)
- `https://artigianoai.it/pt/terms` (PT)

### 2. Add to App Settings

**app/(tabs)/settings/index.tsx:**

```typescript
const TERMS_URLS = {
  it: 'https://artigianoai.it/terms',
  en: 'https://artigianoai.it/en/terms',
  es: 'https://artigianoai.it/es/terms',
  pt: 'https://artigianoai.it/pt/terms',
};

export default function SettingsScreen() {
  const locale = useLocale();

  const openTerms = () => {
    const url = TERMS_URLS[locale] || TERMS_URLS.it;
    Linking.openURL(url);
  };

  return (
    <List.Item
      title="Termini e Condizioni"
      left={() => <List.Icon icon="file-document" />}
      onPress={openTerms}
      right={() => <List.Icon icon="open-in-new" />}
    />
  );
}
```
